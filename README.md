# Ecommerce Website

[![CI](https://github.com/patidarpayal485-pixel/Ecommerce-Wesite/actions/workflows/ci.yml/badge.svg)](https://github.com/patidarpayal485-pixel/Ecommerce-Wesite/actions)

Simple frontend e-commerce demo built with Vite.

## Development

Install dependencies and run dev server:

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- This repository contains only a frontend demo. Serve the built files or use `npm run preview` to locally preview the production build.
